პროექტის სტრუქტურა:

assets - მასში მოთავსებულია ყველა ის სურათი რაც პროექტის აწყობის დროს იყო საჭირო, 
რომელიც თავის მხრივ კიდევ ჩაშლილია რამოდენიმე ფოლდერად, იმის მიხედვით თუ რომელ სექციაში არის ესა თუ ის asset-ი გამოყენებული

data - მასში მოთავსებულია JSON ფაილი courses- სექციისთვის, რაზე დაფუძნებითაც ჯს-ის მეშვობით ვარენდერებ ყველა ქარდს

scripts - მასში მოთავსებულია ყველა ის ჯს ფაილი რომელიც პროექტისთვის არის საჭირო, რომლებიც შემდეგ მთავარ home.js - ფაილში არის დაიმპორტებული და გამოძახებული

styles - არის აწყობილი SASS-ის 7/1 არქიტექტურაზე დაყრდნობით. სადაც მოთავსებულია ყველა ის scss ფაილი რომელიც დამჭირდა პროექტის ასაწყობად და შემდეგ ყველა ეს ფაილი დაიმპორტებულია main.scss-ში

home.html - წერია ის html კოდი რომლითაც შეიქმნა პროექტის



გამოყენებული ტექნოლოგიები: 

HTML - მარქაფისთვის
SCSS - დამატებითი ბენეფიტებისთვის, როგორიცაა ნესტინგი, მიქსინები, ცვლადები და ა.შ.
JS - კომპონენტების გასაცოცლებლად
